# Online vs In-Store Retail Statistical Analysis

## Overview
This project analyzes differences between online and in-store retail performance using statistical inference methods.

## Objectives
- Compare retail performance between online and in-store channels
- Perform hypothesis testing
- Estimate confidence intervals
- Draw statistical conclusions from retail data

## Methods Used
- Hypothesis Testing
- Confidence Intervals
- Two-Sample Statistical Tests
- Data Analysis in R

## Tools & Technologies
- R
- Quarto
- Statistical Inference
- Data Visualization
